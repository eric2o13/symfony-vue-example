# Example project Symfony 6 & VueJS 3

> Note: We are using the VueJS Options API and Vuex



### Installation

```
$ cd example-project
$ composer install
$ yarn install
$ yarn run build
```

### Start dev server

```
$ symfony server:start
```

## Building files
```
$ yarn run build
```

## Watching files
```
$ yarn run watch
```
